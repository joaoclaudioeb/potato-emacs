* Overview
:PROPERTIES:
:CUSTOM_ID: overview
:END:

This repository hosts my Emacs setup, dubbed "The Potato Emacs," implemented as a literate configuration. It is not intended to be an elegant or highly optimized framework.

It is simply my personal, straightforward (and arguably naive) configuration, which fully justifies its "potato" moniker. If you find it useful, you are welcome to use and enhance it.


* Repository organization
:PROPERTIES:
:CUSTOM_ID: repository-organization
:END:

#+begin_src text
potato_emacs/
├── config.org
├── init.el
├── custom.el
└── external/
#+end_src

- =config.org= :: Executable configuration (tangled automatically).  
  Main configuration file (see [[https://github.com/joaoclaudioeb/potato-emacs/blob/main/config.org][potato]]).
- =init.el= :: Minimal bootstrap that loads the tangled configuration.
- =custom.el= :: Variables generated by Emacs Customize.
- =external/= :: Local Emacs Lisp modules (to be deprecated).
