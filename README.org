* Overview
:PROPERTIES:
:CUSTOM_ID: overview
:END:

This repository contains my Emacs configuration (The Potato Emacs) based on *literate programming* using Org mode.


* Repository organization
:PROPERTIES:
:CUSTOM_ID: repository-organization
:END:

#+begin_src text
potato_emacs/
├── config.org
├── init.el
├── custom.el
└── external/
#+end_src

- =config.org= :: Executable configuration (tangled automatically).  
  Main configuration file — see [[file:config.org]].
- =init.el= :: Minimal bootstrap that loads the tangled configuration.
- =custom.el= :: Variables generated by Emacs Customize.
- =external/= :: Local Emacs Lisp modules (to be deprecated).
